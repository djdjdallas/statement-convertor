# Logo Files for Google Workspace Marketplace

## Required Sizes:
- **logo-120x120.png** - OAuth consent screen
- **logo-128x128.png** - Marketplace listing icon
- **logo-256x256.png** - High-res marketplace icon (optional)
- **logo-512x512.png** - Promotional materials (optional)

## Guidelines:
1. Use PNG format with transparent background
2. Ensure logo is centered and has adequate padding
3. Logo should be recognizable at small sizes
4. Follow Google's branding guidelines

## To Generate:
1. Export from your design software (Figma, Sketch, etc.)
2. Or use online SVG to PNG converters
3. Or install canvas package and run: node scripts/convert-logos.js
